# DESIK - YouTube Video Player

Aplikasi Android sederhana untuk memutar video YouTube.

## Fitur
- Putar video YouTube dengan ID video
- 2 video pilihan (tombol Video 1 & Video 2)
- Input manual ID video
- Autoplay
- Dark theme dengan aksen merah

## Cara Pakai
1. Buka aplikasi
2. Pilih Video 1 atau Video 2
3. Atau masukkan ID video manual
4. Tekan PUTAR VIDEO

## Teknologi
- Kotlin
- WebView
- GitHub Actions (auto build APK)
